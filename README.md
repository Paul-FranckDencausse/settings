# Profile Widget

Ce code est une implémentation d'un widget de profil dans une application Flutter. Il affiche les informations de profil telles que le nom, l'e-mail, et propose des options de paramètres de compte telles que le changement de mot de passe et la modification de profil. De plus, il inclut une fonctionnalité de bascule entre le mode sombre et le mode clair de l'application.

## Capture d'écran

![Capture d'écran](screenshot.png)

## Installation

Assurez-vous d'avoir Flutter installé sur votre système.

1. Clonez ce dépôt :

```bash
git clone https://github.com/Paul-FranckDencausse/settings.git
```

2. Accédez au répertoire du projet :

```bash
cd settings
```

3. Exécutez l'application :

```bash
flutter run
```

## Fonctionnalités

- Affichage des informations de profil.
- Option pour changer le mot de passe du compte.
- Option pour modifier le profil.
- Fonction de bascule entre le mode sombre et le mode clair de l'application.

## Dépendances

Ce projet utilise les dépendances suivantes :

- flutter_flow
- provider
- google_fonts

Pour installer ces dépendances, ajoutez-les à votre fichier `pubspec.yaml` et exécutez `flutter pub get`.
